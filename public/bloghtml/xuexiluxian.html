后端学习路线

Part1：一些学习方法/经验

  1. 先掌握、再理解、学习是一个反复的过程，应该由浅入深
  2. 不要只看，要多动手，实践出真知
  3. 可以适当学习一些前端，配合后端一起做，有了成果后会有成就感
  4. 不要把准备工作做的过于复杂，focus在要做的事情上

Part2：学习路线

一、计算机基础

  1. 数据结构与算法
    • 数据结构：链表、栈、队列、树、堆
    • 算法：
      • 排序和查找算法
      • Leetcode 刷题练习
  2. 计算机网络
    • TCP、IP 核心知识点
    • HTTP版本区别、常用请求方法
  3. 操作系统（Linux）
    • 基础命令：ps、cd、top 等
    • 日志查看、异常排查
    • 简单 shell 脚本编写
  4. 设计模式
    • 核心模式：单例模式、工厂模式、策略模式、观察者模式等

二、Java相关

  1. Java基础
    • 面向对象的特性与使用
    • 常用内部集合：ArrayList、HashMap 等
    • 泛型
    • 异常处理
  2. JVM
    • JVM 定义、与Java的关系、跨平台实现原理
    • JVM 内存模型及各区域作用
    • 垃圾回收机制：回收算法、垃圾回收器
    • 类加载过程：加载、链接（验证、准备、解析）、初始化
  3. Java多线程
    • 基本概念：
      • 进程、线程与CPU的关系，多线程解决的问题
      • 线程安全问题：可见性、原子性、有序性
      • Java 线程状态
    • Java中的锁：
      • Syncronized锁
      • ReentrantLock
    • 关键字和线程类使用：
      • Volatile、Syncronized 解决的问题及原理
      • Thread、Callable、Future 的区别和使用
      • JUC 并发集合和原子类：
        • ConcurrentHashMap
        • AtomicInteger

三、相关开发工具

  1. 版本控制工具Git：add、commit、stash、push、pull
  2. Postman（接口测试）
  3. IntelliJ IDEA（开发IDE）
  4. Navicat（数据库管理）
  5. Maven（项目构建）

四、基础框架组件

  1. Spring boot
    • 容器概念
    • 常用注解：@Autowired、@Component、@Service、@Request 等
    • AOP 原理、应用场景及使用方法
    • 实现：启动WEB服务，通过HTTP接口完成简单CRUD程序
  2. MyBatis：掌握简单应用即可

五、数据库

  1. MySQL
    • 基本SQL编写
    • MySQL 底层日志（了解即可）
    • 索引：
      • 类型：Hash索引、二叉树索引、B树、B+树
      • 分类：聚集索引、非聚集索引、联合索引等
    • 锁：
      • 粒度划分：全局锁、表级锁、行级锁
      • 性质划分：共享锁、独占锁
      • 思想划分：乐观锁、悲观锁
    • 事务：
      • ACID 性质
      • 事务隔离级别
      • MVCC 概念及解决的问题（简单了解）
  2. Redis
    • 应用场景与解决的问题
    • 基本数据结构：String、List、Hash、Set、Zset
    • 基础使用：命令行 kv 插入和查询
    • 过期策略、淘汰策略
    • 持久化策略：RDB、AOF
    • 常见场景问题：
      • 缓存穿透、击穿、雪崩
      • 缓存预热和降级
      • 大key、热key解决方案
      • 基于Redis实现分布式锁
      • 缓存更新策略

六、加分的框架组件

  1. 消息队列：Kafka、Rabbit MQ
  2. 性能调优：Arthas
  3. 容器虚拟化技术：Docker & Kubernetes
  4. 监控组件：Prometheus、Grafana

七、其他拓展知识（加分项）

  1. 鉴权：
    • Oauth2.0
    • JWT token
  2. 常见攻防漏洞：
    • XSS
    • CSRF
    • SQL注入
    • DoS和DDoS攻击
  3. HTTPS：
    • 对称加密、非对称加密
    • 签名、证书
  4. 分布式锁
  5. 限流算法
  6. 布隆过滤器
  7. Bitmap原理与应用

Part3：不要投入太多精力的内容

  1. JSP：前后端分离架构下极少使用，有更先进替代技术
  2. Tomcat原理和Servlet
  3. Spring相关（深入源码级）：
    • Spring源码：各种beanfactory的加载原理与机制
    • Spring事务
    • Spring MVC：请求处理流程（DispatcherServlet、HandlerMapping、HandlerAdapter等）
  4. Jetty原理
